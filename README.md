## Project :star2:

This repo contains an UI clone (partial) from Discord website.

<br>

Deployed [here](https://discord.ui-clone.ronne.dev).

<br>

## Techs :rocket:

- [x] [ReactJS](https://reactjs.org);
- [x] [TypeScript](https://www.typescriptlang.org/);
- [x] [Styled Components](https://styled-components.com/).

<br>

## Installation :wrench:

First you need to clone the project using `git clone https://github.com/leoronne/discord-ui-clone.git`.

You can install the application using `npm install` or `yarn install` on the root dir.

<br>

## Start :on:

To start the application interface just run `npm start` or `yarn start` on the root dir.
